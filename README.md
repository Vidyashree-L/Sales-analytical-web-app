

# Retail Sales Analytics Dashboard

## Project Overview
Developed an end-to-end retail analytics solution using Python, SQL, Pandas, and Streamlit.

### Features
- Processed 9,994 retail transactions across 21 columns
- Extracted key business KPIs using Pandas and SQL
- Built interactive Streamlit dashboard
- Dynamic filtering by category, region, and date
- Exported reports to multi-sheet Excel workbooks
- Automated reporting workflow

### Tech Stack
- Python
- Pandas
- SQL
- Streamlit
- OpenPyXL
- Plotly

### KPIs
- Total Sales
- Total Profit
- Average Order Value
- Top Products
- Region-wise Sales
- Monthly Trends
- Customer Metrics

### How to Run

```bash
pip install -r requirements.txt
streamlit run dashboard.py
